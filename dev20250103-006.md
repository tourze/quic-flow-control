# QUIC Flow Control Package 开发文档

## 工作内容概述

### 需求背景
实现QUIC协议的流量控制机制，包括连接级和流级流量控制。

### 核心功能
1. 连接级流量控制
2. 流级流量控制
3. 流量控制窗口管理
4. 阻塞检测和处理

### 依赖关系
- **依赖包**：quic-core, quic-frames
